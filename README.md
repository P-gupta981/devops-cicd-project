# DevOps CI/CD Project 🚀

## Project Overview
This project demonstrates a complete CI/CD pipeline using GitHub, Jenkins, Docker, and Kubernetes.

## Tech Stack
- Node.js
- Docker
- Jenkins
- Kubernetes
- GitHub

## Flow
GitHub → Jenkins → Docker Image → Docker Hub → Kubernetes Deployment

## How to Run Locally

### 1. Install dependencies
npm install

### 2. Run app
node app.js

## Docker Commands

### Build Image
docker build -t devops-app .

### Run Container
docker run -p 3000:3000 devops-app
